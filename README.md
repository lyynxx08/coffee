☕ Coffee Website

A modern, responsive coffee shop website built with clean design principles and smooth user experience. This project represents a stylish online presence for a coffee brand, showcasing products, ambiance, and services.

📌 Features

✅ Fully responsive design (Mobile, Tablet, Desktop)

✅ Modern and minimal UI

✅ Smooth scrolling and clean navigation

✅ Menu section with categorized items

✅ About section with brand story

✅ Contact section with form

✅ Social media integration
